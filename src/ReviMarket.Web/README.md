# ReviMarket.Web — ASP.NET Core MVC

Backend-версия ReviMarket.

## Что уже есть

- ASP.NET Core MVC
- ASP.NET Identity
- Регистрация
- Вход / выход
- Роли: `Customer`, `Creator`, `Admin`
- EF Core + SQLite
- Каталог услуг и товаров
- Биржа заказов
- Исполнители
- Профиль
- Админ-панель `/Admin`
- Заготовка сообщений

## Как запустить локально

```bash
cd src/ReviMarket.Web
dotnet restore
dotnet run
```

База `revimarket.db` создаётся автоматически при первом запуске через `EnsureCreatedAsync()`.

После запуска открыть адрес из консоли, обычно:

```text
https://localhost:5001
```

или

```text
http://localhost:5000
```

## Регистрация

На сайте есть страница:

```text
/Account/Register
```

Типы аккаунта:

- Заказчик — может создавать заказы.
- Исполнитель — может добавлять товары / услуги и откликаться на заказы.

## Как получить доступ к админке

Админом нельзя стать через обычную публичную регистрацию. Это защита от ситуации, когда любой пользователь сам выбирает роль `Admin`.

Порядок:

1. Зарегистрируй обычный аккаунт через `/Account/Register`.
2. Останови сайт.
3. Укажи email этого аккаунта в переменной окружения:

```bash
set Admin__BootstrapEmail=your-email@example.com
```

Для PowerShell:

```powershell
$env:Admin__BootstrapEmail="your-email@example.com"
```

4. Запусти сайт снова:

```bash
dotnet run
```

5. Зайди под этим аккаунтом и открой:

```text
/Admin
```

Если email совпал с зарегистрированным пользователем, при запуске сайт выдаст ему роль `Admin`.
