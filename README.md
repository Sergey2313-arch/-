# ReviMarket

**ReviMarket** — beta-платформа в стиле **black / purple neon**, объединяющая:

- фриланс-биржу;
- заказы для дизайнеров;
- раздел товаров и цифровых услуг;
- демо-чат заказчика и исполнителя;
- чат с поддержкой;
- статистику пользователя и всей платформы;
- оферту и правила площадки;
- developer logs для отладки.

## Статус проекта

Проект находится на предфинишной стадии frontend MVP.

Это пока **frontend beta без backend и настоящей базы данных**. Данные работают в браузере через `localStorage`, а пользователь перед входом принимает или отклоняет сохранение демо-данных.

## Комиссия в demo-оферте

В beta-заготовке указаны такие комиссии:

- заказ / фриланс-сделка — **10%**;
- цифровой товар / шаблон — **7%**;
- срочная сделка / продвижение — **12%**;
- вывод средств — **0%** со стороны платформы.

В текущей beta реальные платежи отключены. Перед реальным запуском оферту, возвраты, комиссии, налоги и обработку персональных данных нужно согласовать с юристом.

## Developer logs

В проект добавлена dev-панель логов:

```text
Ctrl + `
```

Также можно открыть через:

```text
?debug=1
#dev
```

Что логируется:

- запуск приложения;
- статус конфиденциальности;
- переходы между разделами;
- отправка форм;
- клики по важным кнопкам;
- ошибки JavaScript;
- состояние demo-данных в localStorage.

Логи можно:

- открыть в панели;
- очистить;
- экспортировать в JSON;
- смотреть в консоли браузера.

## Исправленные моменты

- стабильнее работает навигация по разделам;
- прямые ссылки типа `#stats`, `#support`, `#offer` корректнее обрабатываются;
