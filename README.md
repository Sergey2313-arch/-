# ReviMarket

**ReviMarket** — beta-платформа в стиле black/purple neon, объединяющая:

- фриланс-биржу;
- заказы для дизайнеров;
- раздел товаров и цифровых услуг.

## Статус

Это первая фронтовая beta-версия без backend и базы данных. Данные работают в браузере через `localStorage`.

## Что есть в beta

- главный экран в новом неоновом стиле;
- демо-авторизация;
- роли: заказчик / исполнитель;
- раздел заказов;
- раздел фриланса;
- раздел товаров;
- список исполнителей по рейтингу;
- создание тестового заказа;
- добавление тестового товара;
- адаптивный интерфейс.

## Технологии

- HTML
- CSS
- JavaScript
- localStorage

## Запуск

Просто открой `index.html` в браузере.

Для runasp.net: загрузи файлы в корень сайта или в папку `wwwroot`.
