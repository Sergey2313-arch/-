# ReviMarket

**ReviMarket** — beta-платформа в стиле **black / purple neon**, объединяющая:

- фриланс-биржу;
- заказы для дизайнеров;
- раздел товаров и цифровых услуг;
- демо-чат заказчика и исполнителя.

## Статус проекта

Проект находится на предфинишной стадии frontend MVP.

Это пока **frontend beta без backend и настоящей базы данных**. Данные работают в браузере через `localStorage`, а пользователь перед входом принимает или отклоняет сохранение демо-данных.

## Что есть в beta

- главный экран в неоновом стиле;
- адаптив под телефон;
- демо-авторизация;
- роли: заказчик / исполнитель;
- раздел заказов для дизайнеров;
- раздел фриланса;
- раздел товаров и услуг;
- список исполнителей по рейтингу;
- создание тестового заказа;
- создание фриланс-заказа;
- добавление тестового товара;
- чат заказчика и исполнителя;
- окно конфиденциальности: принять / отклонить;
- простая очистка пользовательского ввода перед выводом;
- сохранение демо-данных только после согласия.

## Технологии

- HTML
- CSS
- JavaScript
- localStorage

## Запуск

Открой файл:

```text
index.html
```

Для **runasp.net**: загрузи файлы в корень сайта или в папку `wwwroot`.

## Важно по безопасности

В beta-версии нет настоящих паролей и платежей. Для реального запуска нужно добавить backend:

- ASP.NET Identity;
- хеширование паролей;
- роли пользователей;
- SQL Server;
- хранение сообщений на сервере;
- безопасные сделки;
- платежный провайдер;
- нормальную политику конфиденциальности.
