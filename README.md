# ReviMarket

**ReviMarket** — beta-платформа в стиле **black / purple neon**, объединяющая:

- фриланс-биржу;
- заказы для дизайнеров;
- раздел товаров и цифровых услуг;
- категории работ как на крупных биржах;
- предложения исполнителей;
- личный кабинет заказчика / исполнителя;
- демо-чат заказчика и исполнителя;
- чат с поддержкой;
- статистику пользователя и всей платформы;
- оферту и правила площадки;
- developer logs для отладки.

## Статус проекта

Проект находится на предфинишной стадии frontend MVP.

Это пока **frontend beta без backend и настоящей базы данных**. Данные работают в браузере через `localStorage`, а пользователь перед входом принимает или отклоняет сохранение демо-данных.

## GitHub Pages

Проект подготовлен для публикации через **GitHub Pages**.

В репозиторий добавлен workflow:

```text
.github/workflows/pages.yml
```

После включения Pages сайт будет доступен примерно по адресу:

```text
https://sergey2313-arch.github.io/-/
```

Если репозиторий переименовать в `revimarket`, адрес станет лучше:

```text
https://sergey2313-arch.github.io/revimarket/
```

## Как включить GitHub Pages

1. Открой репозиторий на GitHub.
2. Перейди в **Settings**.
3. Открой раздел **Pages**.
4. В **Build and deployment** выбери **GitHub Actions**.
5. Сохрани настройки.
6. Перейди во вкладку **Actions** и дождись успешного деплоя.

## Предложения исполнителей

Раздел **Предложения** — это витрина готовых услуг исполнителей.

Исполнитель может указать:

- название услуги;
- категорию;
- цену;
- срок выполнения;
- описание результата.

В полной версии это станет аналогом карточек услуг с безопасной сделкой, отзывами, рейтингом и оплатой.

## Личный кабинет

В профиле добавлен расширенный кабинет:

- роль аккаунта;
- количество моих предложений;
- demo-стоимость услуг;
- статус конфиденциальности;
- быстрые действия: создать заказ, фриланс-заказ, товар;
- быстрый переход к категориям, предложениям и оферте.

## Комиссия в demo-оферте

В beta-заготовке указаны такие комиссии:

- заказ / фриланс-сделка — **10%**;
- цифровой товар / шаблон — **7%**;
- срочная сделка / продвижение — **12%**;
- вывод средств — **0%** со стороны платформы.

В текущей beta реальные платежи отключены. Перед реальным запуском оферту, возвраты, комиссии, налоги и обработку персональных данных нужно согласовать с юристом.

## Developer logs

В проект добавлена dev-панель логов:

```text
Ctrl + `
```

Также можно открыть через:

```text
?debug=1
#dev
```

Что логируется:

- запуск приложения;
- статус конфиденциальности;
- переходы между разделами;
- отправка форм;
- клики по важным кнопкам;
- ошибки JavaScript;
- состояние demo-данных в localStorage.

Логи можно:

- открыть в панели;
- очистить;
- экспортировать в JSON;
- смотреть в консоли браузера.
